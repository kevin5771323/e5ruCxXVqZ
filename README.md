# 项目简介 java814

本项目是基于Springboot和Vue的电影评论管理系统，采用前后端分离的开发模式，提供了用户和管理员两种角色的登录功能。以下是项目的详细说明。

## 技术栈

- 后端：Java、Springboot、MyBatis、Maven
- 前端：Vue
- 数据库：MySQL

## 系统角色

- 用户：可浏览电影评论，进行评论和评分
- 管理员：管理用户和电影评论，包括增删改查等功能

## 运行环境

- JDK版本：1.8
- Maven版本：建议使用最新版
- MySQL版本：5.7
- Node.js版本：建议使用最新版

## 部署步骤

1. 准备开发环境：安装JDK1.8、Maven、MySQL5.7和Node.js
2. 克隆或下载项目源代码
3. 导入数据库文件至MySQL数据库
4. 使用idea或vscode打开项目，编译并运行后端服务
5. 在前端项目中运行npm install安装依赖，然后运行npm run dev启动前端服务

## 目录结构

详细目录结构如下：

```
- 项目根目录
  - 后端目录（如：src）
    - main
      - java
      - resources
  - 前端目录（如：web）
    - src
    - public
  - 其他文件（如：数据库文件、配置文件等）
```
